## Variable Declaration
Type inference and explicit typing
```crystal
# Type inference
name = "Alice"
age = 30
price = 19.99

# Explicit typing
count : Int32 = 100
message : String = "Hello"

# Multiple assignment
x, y = 10, 20

# Constant
MAX_SIZE = 1000
```
